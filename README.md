# Arduino OLED SPI & Fan

## OLED SPI wiring
| OLED Pin | Arduino Pin | Wire |
| ----------- | ----------- | ----------- |
| Vcc | 5v | rd |
| Gnd | Gnd | bk |
| D0/CLK | 10 | gn |
| D1/MOSI | 9 | bl |
| RES | 13 | pr |
| DC | 11 | or |
| CS | 12 | yl |

## Required Libraries
- Adafruit SSD1306
- Adafruit GFX
- Adafruit BusIO
